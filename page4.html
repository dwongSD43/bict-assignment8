Required:

Event Handler Behaviour

An onload event handler was used on each page to insert the web address of the page in the footer. I was not sure if I needed a separate event handler from the DOM method and the function example but I wanted to make sure I had everything covered and it was good practice.

JavaScript DOM

A built in JavaScript DOM method was used to change the font size on the Welcome page (page 1)

Programmer Defined Function

The Welcome page has a defined function on the change background colour and change font colour buttons.

Accordion

On the About Us page (page 3), the accordion was used to hide and reveal how to contact us. The options are by email, telephone or mail.

CSS Transition

CSS Transition was demonstrated on the Mini Gallery (page 2). A transition was used on the 2 images. The pictures fades in when you mouse over the image and fades to 50% when you mouse out.

Cite:

Event Handler Behaviour

The event handler behaviour I chose to use was onload event. I inseted the url of the page in the footer of the page everytime the the page loads. The two resources I used was w3 Schools onLoad Event and w3Docs How to Get Current URL in JavaScript. I also have onclick behaviour in the Javascript DOM examples and Function examples.

DOM

I learned how to change the font size with DOM at w3Schools JavaScript HTML DOM - Changing CSS

Functions

I learned how to create a custom function from these 2 sandboxes on w3schools Sandbox 1 and Sandbox 2. I also used stackoverflow to find how to pass values to functions from input fields.

Accordian code

I use this site w3 schools How To - Collapsibles/Accordion to help create the accordion. I changed (+) and (-) symbols to arrow heads using this source, How to add an arrow head symbol.

CSS Transition

I used 2 sources to create the CSS transition, w3School CSS Transitions and How to Add a CSS Fade-in Transition Animation to Text, Images, Scroll & Hover. I added a frame and box shadow to the image when you hover over it. I found how to do the box shadow at Box Shadow for Images.

Colour Palette

I selected my colour palette by using colorhunt.co

Google Fonts

The custom fonts I used on this site for the headings can be found at Google Fonts - Lato Font
